Language Translator

A modern language translation web application developed using HTML, CSS, and JavaScript. The project provides real time translation between 130+ languages with features like language detection, speech playback, copy tools, and a responsive user interface.

This project was built to explore the practical implementation of translation systems, browser APIs, and interactive frontend design while creating a clean and userfriendly translation experience.

Overview

The system allows users to enter text, select source and target languages, and instantly translate content using the Google Translate API. It also includes additional utilities such as text-to-speech, clipboard support, and language swapping for a smoother workflow.

The project demonstrates how frontend technologies and APIs can be integrated to create realworld language processing applications.
